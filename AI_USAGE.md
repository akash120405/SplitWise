# AI_USAGE.md

AI Tools Used -  ChatGPT (GPT-5), cursor, GitHub Copilot, claude(for reference/clarification) 

ChatGPT, cursor, GitHub Copilot was used as a supporting tool during the development of this project. It helped generate initial ideas, explain concepts, suggest implementations, and assist in debugging issues encountered during development.

Some of the areas where ChatGPT/cursor/GitHub Copilot was used include:

* Creating the initial Prisma database schema
* Generating Express.js route templates
* Implementing CSV file upload and parsing functionality
* Assisting with JWT authentication setup
* Suggesting the expense settlement algorithm
* Helping write project documentation
* Troubleshooting errors and debugging code

The AI-generated code was never used directly without review. All suggestions were checked, tested, and modified where necessary before being included in the final project.

## Examples of Incorrect AI Suggestions

Although ChatGPT was helpful, some of its suggestions were not completely accurate and required corrections.

### Case 1: Prisma Relation Handling

#### AI Suggestion

data: {
  groupId,
  paidById
}
#### Issue

While creating an expense, Prisma returned the following error:

Argument group is missing

 How It Was Found

The error appeared during runtime testing of the expense creation API.

#### Fix Applied

The Prisma schema and generated client were reviewed, and the relation fields were updated according to the actual schema requirements.

#### Takeaway

AI-generated database code should always be verified against the schema and generated Prisma client.

---

### Case 2: CSV Amount Parsing

#### AI Suggestion

Number(row.amount)

#### Issue

CSV values such as:
1,200
were converted to:
NaN

#### How It Was Found

Expense records failed to be created during CSV imports.

#### Fix Applied

The input value was cleaned before conversion:

Number(String(row.amount).replace(/,/g, ""))

#### Takeaway

Real-world CSV data often requires preprocessing and validation before use.

---

### Case 3: Incorrect Import Statement

#### AI Suggestion

ChatGPT generated code that assumed a function was exported differently than it actually was.

#### Issue

The application produced the following runtime error:

calculateGroupBalances is not a function

#### How It Was Found

The settlement endpoint failed during testing.

#### Fix Applied

The module exports were checked and the import statement was corrected.

#### Takeaway

Generated imports and exports should always be verified manually.

---

### Case 4: Route Configuration Error

#### AI Suggestion

Some generated route examples caused authentication routes and group routes to become mixed during implementation.

#### Issue

Requests to:
POST /api/auth/register

returned:
Cannot POST /api/auth/register
#### How It Was Found
The issue was identified while testing APIs using Postman.

#### Fix Applied
Route files were reorganized and route registration in `app.ts` was corrected.

#### Takeaway
API routes should always be tested after integration to ensure proper registration.

## Validation Process

Every AI-generated suggestion was reviewed before use. The project was validated using:

* TypeScript compiler checks
* Prisma validation and migrations
* Runtime testing
* Postman API testing
* Database verification through Prisma Studio

Any code generated by AI was treated as a starting point and was only included after successful testing.

---

## Overall Assessment

ChatGPT/Cursor/Claude/GitHub Copilot was a useful development assistant throughout the project. It helped reduce development time, provided implementation ideas, and assisted in debugging several issues. However, not every suggestion was correct, and some required significant modifications before they could be used.

The final project is the result of manual development, testing, and verification, with AI serving as a supporting tool rather than a replacement for developer decision-making. All major implementation decisions and final code changes were reviewed and validated manually.
